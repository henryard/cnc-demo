基础3B 程序格式结构
BXBYBJGZ
B分隔符（固定）
XX 坐标值
YY 坐标值
J计数长度
G计数方向
Z加工指令
说明各字段详解
分隔符 B
固定写法，将 X、Y、J 三个数值分隔开，共写 3 个 B，不可省略
X / Y 值
坐标绝对值，单位 μm（微米）。数值为 0 时可省略不写，但 B 不能省
计数长度 J
直线/斜线：终点在计数方向上的坐标绝对值；圆弧：各象限投影长度之总和
计数方向 G
GX 或 GY，通过比较终点坐标大小决定，影响加工精度
加工指令 Z
L1–L4（直线/斜线）、SR1–SR4（顺时针圆弧）、NR1–NR4（逆时针圆弧）

类型一直线编程

起点在坐标原点，沿 X 轴或 Y 轴方向的直线。X、Y 值均为 0，可省略不写；计数长度 J 为终点坐标绝对值。

示例 1：沿 +Y 方向走 15.8 mm（即 15800 μm）
B B B 15800 GY L2
X=0 省略，Y=0 省略，J=15800，计数方向 GY
示例 2：沿 -X 方向走 20.0 mm（即 20000 μm）
B B B 20000 GX L3
X=0 省略，Y=0 省略，J=20000，计数方向 GX
⚠ B 分隔符必须写 3 个，即使 X 和 Y 省略，格式仍为 BBB[J][G][Z]
类型二斜线编程

有斜率的直线。起点默认为原点，X、Y 为终点坐标绝对值。

计数方向选取规则（比较终点坐标绝对值）

条件取计数方向J 值
|Xe| > |Ye|GX终点 X 坐标绝对值
|Xe| < |Ye|GY终点 Y 坐标绝对值
|Xe| = |Ye|GX 或 GY 均可两者相等，取其一
示例：终点 A(12, 8) → |12| > |8| → 取 GX，J=12000
B 12000 B 8000 B 12000 GX L1
斜线偏向 +X 方向，故取 GX，加工指令 L1
类型三圆弧编程

坐标原点为圆心，X、Y 为圆弧起点坐标绝对值。计数长度为各象限投影总和。

圆弧计数方向（与斜线规则相反！）

条件（终点坐标）取计数方向
|Xe| < |Ye|GX
|Xe| > |Ye|GY
|Xe| = |Ye|GX 或 GY 均可
⚠ 圆弧选 G 方向与斜线规则完全相反（< 取 GX，> 取 GY），最易出错！
示例：逆圆弧，起点 A(−2, 9)，终点 B(9, −2)
B 2000 B 9000 B 25440 GY NR2
终点 |9| > |−2|，取 GY；J=投影之和=25440μm
类型四连续图形（矩形）

将多条直线或圆弧指令连接起来形成封闭图形。最后一段必须加 DD 全停机。

示例：20mm × 10mm 矩形（起点左下角，逆时针）
段1 B B B 20000 GX L1
// 沿 +X 走 20mm (底边)
段2 B B B 10000 GY L2
// 沿 +Y 走 10mm (右侧边)
段3 B B B 20000 GX L3
// 沿 -X 走 20mm (顶边)
段4 B B B 10000 GY L4 DD
// 沿 -Y 走 10mm (左侧边) 并停机
流程开机加工标准步骤
1
将空气开关往上拨，接通三相电源
检查：急停按钮未按下，超行程触点未压下
2
输入 3B 程序代码到控制器
待命状态（显示P）→ 按起始段号数字键 → 逐条输入
3
按下启动按钮
注意：前门门控开关打开时启动按钮失效
4
手动碰火花校正钼丝垂直度
上丝用低速；正常切割用高速，调试完成后需关闭调试模式
5
开启水泵，确认喷水状态正常
喷水需包住钼丝，水量不足会引起断丝
6
执行加工：待命 → 起始段号 → 执行 → 执行
状态控制器开机状态判断
显示 "Good"（正常状态）▶
RAM 内部正常，停电保护有效。可直接输入程序或继续上次操作。
显示 "Good" 但喇叭鸣叫闪烁▶
RAM 状态不正常，程序数据可能有误或丢失。建议重新输入完整程序。
显示 "Error"（出错状态）▶
加工过程中意外掉电后开机出现，RAM数据损坏。需重新输入全部程序。
显示掉电前正在加工的状态▶
数据完好保留。开动机床即可从断点继续正常切割。

功能一间隙补偿与校零

封闭图形才能做补偿，最后一条指令必须加停机符。

置入：待命 → 上档 → 补偿量 → 按 X(正) 或 Y(负)

校零：待命 → 上档 → 起始段号 → 校零 → 等待 → 校零

负号规定：逆时针编程时，割凸模补偿量取正，割凹模取负；顺时针相反。
功能二旋转与缩放
旋转加工
多齿轮等对称零件。操作：上档 → 起止段号 → 次数 → 旋转 → 整数 → 旋转 → 小数 → 旋转。
缩放加工
1000为基数，1005即放大5‰。置入：上档 → 比例值 → 换档＋X/Y。
功能三平移加工

同一段程序重复执行多次，适用于批量加工。

示例：第8条到第88条平移18次
上档 → 按 8 → (…) → 按 88 → (…) → 按 18 → 平移 → 待命

故障钼丝与加工问题
容易断丝 / 跳动厉害▶
检查导轮V形槽是否磨损，轴承是否损坏。必要时更换钼丝或导轮组件。
导电块冒火花 / 未开高频断丝▶
导电块磨损产生凹槽，或功放管击穿。打磨更换导电块，排查损坏的功放管。
切割面光洁度差▶
可能是乳化液变质、跟踪调节不当、钼丝张力不均。需重新张紧并更换工作液。
