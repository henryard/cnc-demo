说明各字段详解
分隔符 B
固定写法，将 X、Y、J 三个数值分隔开，共写 3 个 B，不可省略
X / Y 值
坐标绝对值，单位 μm（微米）。数值为 0 时可省略不写，但 B 不能省
计数长度 J
直线/斜线：终点在计数方向上的坐标绝对值；圆弧：各象限投影长度之总和
计数方向 G
GX 或 GY，通过比较终点坐标大小决定，影响加工精度
加工指令 Z
L1–L4（直线/斜线方向）、SR1–SR4（顺时针圆弧）、NR1–NR4（逆时针圆弧）

类型一直线编程

起点在坐标原点，沿 X 轴或 Y 轴方向的直线。X、Y 值均为 0，可省略不写；计数长度 J 为终点坐标绝对值。

示例 1：沿 +Y 方向走 15.8 mm（即 15800 μm）
B  B  B  15800  GY  L2
X=0 省略，Y=0 省略，J=15800，计数方向 GY（与 L2 对应的 +Y 轴方向一致）
示例 2：沿 -X 方向走 20.0 mm（即 20000 μm）
B  B  B  20000  GX  L3
X=0 省略，Y=0 省略，J=20000，计数方向 GX（与 L3 对应的 -X 轴方向一致）
⚠ B 分隔符必须写 3 个，即使 X 和 Y 省略，格式仍为 BBB[J][G][Z]
类型二斜线编程

有斜率的直线（既不平行也不垂直于坐标轴）。起点默认为原点，X、Y 为终点坐标绝对值。

计数方向选取规则（比较终点坐标绝对值）

条件取计数方向J 值
|Xe| > |Ye|GX终点 X 坐标绝对值
|Xe| < |Ye|GY终点 Y 坐标绝对值
|Xe| = |Ye|GX 或 GY 均可两者相等，取其一
示例：终点 A(12, 8) → |12| > |8| → 取 GX，J=12000
B  12000  B  8000  B  12000  GX  L1
起点在原点，终点 (12000, 8000) μm，斜线偏向 +X 方向，故取 GX，加工指令 L1
类型三圆弧编程

坐标原点为圆心，X、Y 为圆弧起点坐标绝对值。计数长度为各象限圆弧在计数方向投影的总和。

圆弧计数方向（注意：与斜线规则相反！）

条件（终点坐标）取计数方向
|Xe| < |Ye|GX
|Xe| > |Ye|GY
|Xe| = |Ye|GX 或 GY 均可
⚠ 圆弧选 G 方向与斜线规则完全相反（< 取 GX，> 取 GY），这是最易出错的地方！
示例：逆圆弧，起点 A(−2, 9)，终点 B(9, −2)，R≈9.22
B  2000  B  9000  B  25440  GY  NR2
终点 |9| > |−2|，取 GY；J=各象限在GY投影之和=25440μm；NR2=逆圆从第二象限起
类型四连续图形（矩形）编程

将多条直线或圆弧指令连接起来形成封闭图形。最后一段指令后必须加 DD 表示全程序结束。

示例：切割一个 20mm × 10mm 的矩形（起点为左下角，逆时针切割）
段1 B B B 20000 GX L1 // 沿 +X 走 20mm (底边)
段2 B B B 10000 GY L2 // 沿 +Y 走 10mm (右侧边)
段3 B B B 20000 GX L3 // 沿 -X 走 20mm (顶边)
段4 B B B 10000 GY L4 DD // 沿 -Y 走 10mm (左侧边) 并停机
引线切割说明：实际加工中，为避免在工件轮廓上留下起步痕迹，通常会从图形外的穿丝孔开始，先切一条引线进入图形轮廓，最后再切回穿丝孔。
要点完整程序编写注意事项

① 每条指令固定由 3 个 B 分隔 X、Y、J 三段数据，格式不可改变

② 段末必须写停机符 D，整个程序最后一段写 DD（全部停机）

③ 封闭图形作间隙补偿时，第一条须是引线并加 L，最后一条方向相反加 D 或 DD

④ 坐标值为 0 可省略，但 B 分隔符绝对不能省略

⑤ 坐标单位统一为 μm（微米），1 mm = 1000 μm

流程开机加工标准步骤
1
将空气开关往上拨，接通三相电源
检查：急停按钮未按下，超行程触点未压下，否则启动失效
2
输入 3B 程序代码到控制器
待命状态（显示P）→ 按起始段号数字键 → 按3B格式逐条输入 → 末段必须输入DD
3
按下启动按钮
注意：前门门控开关打开时启动按钮失效，请关好前门
4
按下开运丝按钮，手动碰火花校正钼丝垂直度
上丝用低速；正常切割用高速。调试LED亮表示调试模式，上丝完成后需关闭
5
按下开水泵按钮，确认喷水状态正常
喷水以包住钼丝为原则；水量不足或水路不畅会引起断丝
6
执行加工：待命 → 起始段号 → 执行 → 执行
显示器左侧为起始地址，右侧为结束地址，再按执行正式开始加工
上电控制器开机状态判断
显示 "Good"（正常状态）▶
RAM 内部正常，停电保护有效。可直接输入程序或继续上次操作，是最理想的开机状态。
显示 "Good" 但喇叭鸣叫 + 屏幕闪烁▶
RAM 状态不正常，程序数据可能有误或丢失。建议重新输入完整程序后再开始加工。
显示 "Error"（出错状态）▶
加工过程中意外掉电后开机出现此状态，RAM中的指令很可能已损坏。需重新输入全部程序再加工。
显示掉电前正在加工的状态▶
数据完好保留。只要开动机床，就可以从断点继续正常切割，无需重新输入程序。

功能一间隙补偿与校零

封闭图形才能做补偿，最后一条指令必须加停机符，否则无法切割或产生错误。

置入补偿值：待命 → 上档 → 输入补偿量数字 → 按 X（正值/凸模逆时针）或 Y（负值/凹模逆时针）

校零验证：待命 → 上档 → 起始段号 → 校零 → 等待段号停止跳动 → 再按校零 → X轴误差=0，Y轴误差=0则正确

负号规定：逆时针编程时，割凸模补偿量取正，割凹模取负；顺时针编程时相反。
补偿值范围：±1 至 ±9999 μm（四位十进制数）
功能二旋转加工

适用于多齿轮、多孔等旋转对称零件，角度精度保留小数点后 6 位。

操作步骤示例：从第22条到第100条旋转12.345678°，共1次
上档 → 按 22 → (…) → 按 100 → (…) → 按 1（次数）→ 旋转
→ 按 12（整数部分）→ 旋转 → 按 345678（小数部分）→ 旋转 → 待命
⚠ 控制器内只能有一项旋转参数，再输入会覆盖前一项。执行加工时不能修改旋转参数。
功能三缩放加工

以 1000 为基数，每增加 1 放大 1‰，每减少 1 缩小 1‰。

输入参数效果说明
1005放大 5‰比原图大 0.5%
995缩小 5‰比原图小 0.5%
1000不缩放原尺寸切割
置入：上档 → 缩放参数值 → 换档＋X 或 Y。加工碰到 DD 后自动清除缩放参数，只能使用一次。
功能四平移加工

对同一段程序重复执行多次，适用于批量相同形状加工。

示例：第8条到第88条平移18次
上档 → 按 8 → (…) → 按 88 → (…) → 按 18 → 平移 → 待命
此设置将第8–88条加工完毕后再重复加工18次，共19次
清除平移：上档 → 平移 → D → 显示P，清除完成，指示灯灭。

断丝类钼丝相关故障
钼丝跳动厉害，容易断丝▶
可能原因：钼丝粗细不均；导轮V形槽磨损；导轮轴承损坏（按住导轮座抖动大、噪声大）
处理：更换钼丝；更换导轮；更换导轮轴承
火花时有时无，易短路/开路▶
可能原因：跟踪速度未调好；工作液太脏或水路不畅；工件材质差，有焊渣等杂质
处理：调整跟踪速度；更换工作液；疏通水路；检查工件材质
导电块冒火花，或未开高频也断丝▶
可能原因：导电块、挡丝棒上有凹槽（长期磨损形成）；功放管击穿（碰工件出现弧光电流很大）
处理：拆下打磨或更换导电块；★更换已坏功放管
断丝保护不正常（换向时停机/断丝不停机）▶
换向时停机：钼丝太松与导电块接触不良，重新紧好钼丝。
钼丝断了却不停机：KA3 继电器损坏，★需更换 KA3。
电源类电源与电机故障
总电源不能开启▶
可能原因：限位器碰到超行程保护开关SQ3；SB1/SB3急停按钮自锁；电源缺相；KM1损坏
处理：移开限位器；解除自锁；恢复电源；★更换KM1
水泵电机不能开启▶
按手控盒开水按钮后 KM3 不吸合：变频器故障或 KM3 损坏。
KM3 吸合但水泵不转：380V 供电线断路或水泵电机烧坏。
处理：★维修变频器；★更换KM3；★接通供电线；★更换水泵
运丝电机不能开启 / 不换向▶
不能开启：SB5按钮损坏或变频器故障。
不换向：行程开关SQ1/SQ2触点不通或变频器故障。
处理：★更换SB5；★维修变频器；★更换行程开关
电机类步进电机故障
步进电机锁不住▶
一个轴锁不住：连接该步进电机的导线断，或对应功放管烧坏。
两个方向都锁不住：驱动电源保险丝断。
处理：★接通线路；★换损坏元器件；★更换保险丝
步进电机失步 / 运行抖动▶
快速运动时失步：速度设置过快，重新调慢；或驱动滤波电容容量不足。
运行抖动：驱动线路缺相损坏或驱动信号不正常，指示灯常亮/常灭时对应功放管坏。
处理：重新调整速度；★更换滤波电容；★更换已坏元器件/功放管
加工类加工质量问题
无火花 / 高频打不开▶
检查：①控制台高频开关是否已打开 ②运丝电机是否已开（未开时高频无法输出）③高频电源是否故障 ④电压/电流表是否开路
切割面光洁度差▶
可能原因：导轮/轴承磨损有跳动；运丝丝杆与螺母有间隙；乳化液变质或配比不合适；跟踪调节过紧或过松；钼丝张力不均（一边松一边紧）；工件材质差
处理：更换磨损件，更换工作液，重新调整跟踪速度，均匀张紧钼丝
加工不稳定（倒计数字忽快忽慢）▶
可能原因：乳化液太脏；功放管偏少；跟踪调节不好；取样线接触不良；导轮及轴承故障；钼丝过松；高频脉冲间隔太小，排屑困难
处理：更换工作液，增加功放管，调整跟踪，检查取样线和导轮
钼丝钼丝使用寿命参考
丝径状态建议操作
新 18 丝≤ 3.5A 小电流切割；切割 10000 mm² 后或丝面发白后，可转大电流
磨损至 16 丝将丝筒两侧钼丝分别向内侧偏移，防止换向时卡在加工间隙中断丝
磨损至 13 丝建议更换新钼丝；过细会导致光洁度下降，大电流下极易断丝
经验数据：18丝新钼丝切割60mm钢材（5A电流），平均效率约8000 mm²/h，累计可切割约 350000 mm²（累计40小时以上）。
寿命应以切割面积计算，而非使用时间。
